# Privacy Policy

PixelPeek is designed to be local, lightweight, and private.

## Data Collection

PixelPeek does not collect personal data.

PixelPeek does not:

- track browsing history
- send page content to external servers
- use analytics
- use advertising scripts
- sell or share user data
- require login or account creation

## Local Settings

PixelPeek stores extension preferences through Chrome extension storage, including:

- whether PixelPeek is enabled
- whether natural image size is shown
- whether rendered image size is shown
- whether background image detection is enabled

These settings are used only to control extension behavior.

## Network Requests

PixelPeek does not make external API requests.

When background image detection is enabled, the browser may load the referenced background image URL in order to read its dimensions. This happens locally in the user's browser as part of normal webpage behavior.

## Contact

For questions or issues, open a GitHub issue in the PixelPeek repository.
